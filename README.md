# ARISE Church Dunedin Documentation

A comprehensive documentation website for ARISE Church Dunedin, built with [Nextra](https://nextra.site/) and powered by [Vercel](https://vercel.com/).

## Overview

This documentation site provides detailed operational guides for various ministry teams at ARISE Church Dunedin, including:

- **AV Team** - Complete audio-visual operations guide with hardware, setup, operation, and troubleshooting
- **Sound Team** - Audio system documentation (coming soon)
- **Packdown Team** - Setup and breakdown procedures (coming soon)

## Features

- 📱 **Responsive Design** - Optimized for desktop, tablet, and mobile
- 🔍 **Full-Text Search** - Quickly find information across all documentation
- 🎨 **ARISE Church Branding** - Styled to match the main church website
- 📖 **Comprehensive AV Guide** - Step-by-step procedures and troubleshooting
- 🔧 **Easy Navigation** - Organized sidebar with quick access to all sections
- 💬 **Feedback System** - GitHub integration for suggestions and improvements

## Development

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Bakhombisile02/AriseDunedinDocs.git
cd AriseDunedinDocs
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
```

## Deployment

This site is designed to be deployed on [Vercel](https://vercel.com/). Simply:

1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect the Next.js framework
3. Deploy with default settings

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Bakhombisile02/AriseDunedinDocs)
