## EasyWorship and Stream Deck Operation

### Initial Software Setup

<Steps>

### Launch EasyWorship
1. Turn on the AV1 PC
2. Open EasyWorship from desktop or start menu
3. Load the current service schedule
4. Verify song and sermon slide presentations are ready

### Sync Stream Deck
1. Check for small warning icons on Stream Deck buttons
2. Press the button with the **EasyWorship icon**
3. Warning icons should disappear
4. Test button responsiveness

</Steps>

<Callout type="info" emoji="🎛️">
For Stream Deck shortcuts to work, EasyWorship must be the active, focused window on the computer.
</Callout>

### Operating Songs During Service

**Stream Deck Controls:**
- **"Chorus" Button** - Jumps to chorus slide in current song
- **"Bridge" Button** - Jumps to bridge slide in current song
- **Next/Previous** - Navigate through song slides
- **Clear** - Remove graphics from display

**Manual Operation:**
- Use EasyWorship interface for detailed control
- Click slides directly to display them
- Use preview window to prepare next slide
- Monitor both main and parents' room outputs

### Best Practices for Service Operation

**Pre-Service Preparation:**
1. Load complete service schedule in EasyWorship
2. Test all slides and media elements
3. Verify Stream Deck shortcuts are working
4. Set up camera framing and focus
5. Check audio levels with sound team

**During Service:**
- Keep EasyWorship as the active window
- Use Stream Deck for quick navigation
- Monitor video quality on all outputs
- Coordinate with worship leader for slide timing
- Be ready for spontaneous song changes

**Troubleshooting Live Issues:**
- If Stream Deck becomes unresponsive, click EasyWorship icon
- For presentation issues, use manual EasyWorship controls
- If video quality degrades, check cable connections
- Communicate issues immediately to team leader

## ATEM Switcher Operation

### Understanding the Control Surface

**Program Row (Red):** Currently live video output
- Camera input typically labeled "1" or "CAM"
- PC graphics typically labeled "2" or "PC"
- Only one button lit red at a time

**Preview Row (Green):** Next source to go live
- Shows what will be live after transition
- Use for preparation and cueing

**Transition Controls:**
- **CUT Button** - Immediate switch between sources
- **AUTO Button** - Smooth transition with effects
- **T-Bar** - Manual transition control

### Common Switching Operations

**Camera to Graphics:**
1. Set graphics source in Preview (green)
2. Press CUT or AUTO to transition
3. Verify graphics are displaying correctly

**Graphics to Camera:**
1. Set camera source in Preview
2. Check camera framing and focus
3. Transition using CUT or AUTO

**Emergency Situations:**
- BLACK button immediately cuts to black output
- Use for technical difficulties or sensitive moments
- Always have backup plan for equipment failure

## Resi Streaming Operations

Use these procedures when connecting to services from other ARISE campuses.

<Callout type="info" emoji="📡">
Resi streaming allows us to receive and broadcast services from other locations.
</Callout>

### Connecting to Remote Campus

<Steps>

### Select Event
1. Using Resi monitor and keyboard
2. Navigate to **Events** tab in left-hand menu
3. Look for current day's service listing

### Choose Service
1. Select appropriate event (e.g., "ARISE Centre Morning")
2. Click **"Select Event"** button
3. Wait for connection to establish

### Monitor Stream
1. Navigate to **Cues** tab
2. Review timeline showing pre-programmed actions
3. Monitor connection status at bottom of screen
4. Note any scheduled videos or special elements

### Go Live
1. Stream will begin buffering automatically
2. Monitor status indicators
3. Verify audio and video quality
4. Communicate with local sound team for audio levels

</Steps>

### Stream Monitoring

**Quality Indicators:**
- **Green Status** - Excellent connection
- **Yellow Status** - Good connection with minor issues
- **Red Status** - Poor connection, consider backup plans

**During Remote Service:**
- Monitor stream continuously
- Be prepared to switch to local backup
- Coordinate with hosting campus if issues arise
- Maintain communication with local team leaders

## Live Service Coordination

### Communication Protocols

**With Worship Team:**
- Use hand signals for slide changes
- Establish eye contact before transitions
- Have backup plans for technical issues
- Be flexible for spontaneous changes

**With Sound Team:**
- Coordinate audio levels for all outputs
- Ensure parents' room audio is appropriate
- Monitor for audio/video sync issues
- Share audio routing information

**Emergency Procedures:**
- Know location of manual overrides
- Have contact information for technical support
- Understand backup presentation options
- Maintain calm professional demeanor

### Service Flow Management

**Pre-Service (30 minutes before):**
- Complete all setup procedures
- Test all systems thoroughly
- Load service schedule in EasyWorship
- Coordinate with all team members

**During Service:**
- Follow service order closely
- Anticipate slide and video needs
- Monitor all output displays
- Be ready for unplanned elements

**Post-Service:**
- Safely shut down equipment in reverse order
- Report any technical issues
- Store equipment properly
- Complete setup checklist for next service
